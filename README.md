# AA Techs Website

Professional website for AA Techs - providing managed IT services, DevOps solutions, and scalable systems for growing businesses.

## About AA Techs

We support Startups, Remote Teams, & Growing Businesses that need dependable, forward-thinking tech support. Our services include:

- **Managed IT**: Full support and responsibility for your business technology
- **DevOps**: Helping your developers move faster, deploy smarter
- **Scalable Systems**: Infrastructure that adapts as your business grows

## Website Features

- Responsive design optimized for all devices
- Modern HTML5/CSS3/JavaScript architecture
- Bootstrap framework for responsive layouts
- GSAP animations for smooth user experience
- Contact forms and newsletter subscription
- Partner showcase and service highlights

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **CSS Framework**: Bootstrap 5
- **Animations**: GSAP (GreenSock)
- **Icons**: Font Awesome, Line Awesome, Iconoir
- **Fonts**: Google Fonts (DM Sans, Syne, Yantramanav)
- **Build**: Static HTML (no build process required)

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/reddomeuk/aatechs-website.git
   cd aatechs-website
   ```

2. Install dependencies (optional, for local server):
   ```bash
   npm install
   ```

3. Start local development server:
   ```bash
   npm run dev
   ```
   
   Or simply open `index.html` in your browser.
